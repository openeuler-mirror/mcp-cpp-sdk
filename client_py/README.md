# MCP FileSystem Server IPC Demo

这个目录包含用于测试MCP文件系统服务器IPC功能的示例文件和目录。

## 传输方式

- **IPC (进程间通信)**: 通过管道或共享内存进行通信
- **优势**: 低延迟、高安全性、适合本地通信
- **适用场景**: 本地工具、开发环境、单机部署

## 目录结构

- `test_files/`: 测试文件目录
  - `sample.txt`: 示例文本文件
  - `subdir/`: 子目录
- `docs/`: 文档目录
- `logs/`: 日志目录
- `README.md`: 本说明文件

## 测试方法

1. 使用 `list_directory` 工具列出目录内容
2. 使用 `read_file` 工具读取文件内容
3. 使用 `write_file` 工具创建新文件
4. 使用 `create_directory` 工具创建新目录
5. 使用 `file_info` 工具获取文件信息

## IPC 通信特点

- 进程间直接通信，无需网络
- 数据安全性高，不会泄露到网络
- 响应速度快，延迟低
- 适合本地开发和测试环境
